# TodoList-Django
A Todo List Website that helps you create, read, update and delete tasks.

# Tech Stacks Used
* Python
* Django
* HTML
* CSS

# How to run the application
* Clone the repo
* Setup Django server
* Execute "python manage.py runserver" to start the server
* "python manage.py migrate" for migrations
